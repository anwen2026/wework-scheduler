# 企业微信排单助手 - 安装和使用指南

## 📦 安装步骤

### 1. 准备工作

**你需要：**
- ✅ 红米 Note 9 5G（Android 12）
- ✅ 已安装 LSPosed 框架
- ✅ 已安装企业微信

### 2. 编译 APK

**在电脑上操作：**

```bash
# 1. 进入项目目录
cd /Users/mr.yao/.openclaw/workspace/wework-scheduler

# 2. 编译 APK
./gradlew assembleDebug

# 3. APK 位置
# app/build/outputs/apk/debug/app-debug.apk
```

### 3. 安装到手机

```bash
# 通过 ADB 安装
adb install app/build/outputs/apk/debug/app-debug.apk

# 或者直接复制到手机安装
```

### 4. 在 LSPosed 中激活

1. 打开 LSPosed Manager
2. 找到"企业微信排单助手"
3. 勾选"企业微信"
4. 重启企业微信

## 🎯 使用指南

### 基本流程

```
1. 在企业微信创建一个"排单群"
2. 在群里发送指令创建任务
3. 发送要排单的消息
4. 到时间自动发送
```

### 指令列表

| 指令 | 说明 | 示例 |
|------|------|------|
| `hwsd HH:MM` | 创建排单任务 | `hwsd 18:00` |
| `停止` | 暂停排单 | `停止` |
| `开始` | 恢复排单 | `开始` |
| `取消任务` | 取消整个任务 | `取消任务` |
| `状态` | 查看任务状态 | `状态` |
| `撤销码` | 撤销指定消息 | `W4SDH56YK7MN3PQ` |
| `延迟N分钟` | 延迟任务 | `延迟10分钟` |
| `提前N分钟` | 提前任务 | `提前5分钟` |
| `间隔N秒` | 修改消息间隔 | `间隔60秒` |
| `帮助` | 查看帮助 | `帮助` |

## ⚙️ 配置

### 设置排单群

1. 打开"企业微信排单助手" App
2. 点击"打开设置"
3. 添加排单群 ID

### 设置发单方案

1. 创建发单方案（定义要发送到哪些群）
2. 设置为默认方案
3. 排单时会自动使用该方案

### 修改默认间隔

默认消息间隔是 30 秒，可以修改：

```
你：间隔60秒
机器人：✅ 已修改消息间隔为 60 秒
```

## 🐛 故障排除

### 模块没有生效

1. 检查 LSPosed 是否激活模块
2. 检查是否勾选了"企业微信"
3. 重启企业微信
4. 查看 LSPosed 日志

### 消息没有自动发送

1. 检查任务状态（发送"状态"指令）
2. 检查手机是否锁屏（WorkManager 可能被系统限制）
3. 检查企业微信是否在后台运行
4. 查看日志：`adb logcat | grep WeWorkScheduler`

### 撤销码无效

1. 确认撤销码是 15 位字母数字组合
2. 确认消息还没有发送
3. 已发送的消息无法撤销

## 📝 注意事项

1. **排单群和目标群要分开**：排单群用于发送指令，目标群用于接收消息
2. **消息间隔不要太短**：建议至少 10 秒，避免被企业微信限流
3. **定时任务依赖系统**：如果手机省电模式过于激进，可能影响定时发送
4. **数据本地存储**：所有数据存在手机本地，卸载 App 会丢失
5. **仅供学习使用**：请勿用于违法违规用途

## 🔧 高级功能

### 批量排单

可以一次性排很多消息，系统会自动计算每条消息的发送时间。

### 多个任务

可以同时创建多个任务（不同时间），系统会自动管理。

### 任务历史

所有任务都会保存历史记录，可以查看。
